osm-to-3dm converter

A tiny utility that turns OpenStreetMap building footprints into Rhino .3dm volumes.

Drop an OpenStreetMap .osm file onto the browser converter to generate a Rhino .3dm model instantly. Everything stays on your device — no uploads, no installs.

Convert directly in your browser

  1. Export an .osm file that includes building or building:part geometry from openstreetmap.org.
  2. Open the web converter & viewer and drop the file onto the page.
  3. Inspect the generated building volumes immediately in 3D and use the “Download 3DM” button to save the Rhino model.

The conversion runs entirely in your browser using WebAssembly — large files never leave your machine.

Prefer the command line?

  1. Install the rhino3dm Python bindings: 
    pip install rhino3dm
  2. Export an .osm file with the buildings you care about.
  3. Run the converter script: 
    python osm_to_3dm.py input.osm output.3dm

Flags such as --default-height and --level-height let you tweak how missing OSM attributes are interpreted.

Troubleshooting GitHub Pages

To make sure this page shows up at https://<username>.github.io/osm-to-3dm/:

  1. In your repository settings, open the “Pages” section.
  2. Under Build and deployment, choose Deploy from a branch.
  3. Pick the main branch and the / (root) folder, then hit Save.
  4. GitHub rebuilds the site and serves this index.html. That usually takes less than a minute.

If you already had Pages enabled, trigger a rebuild by making a fresh commit or using the “Retry” button inside the Pages settings panel.
