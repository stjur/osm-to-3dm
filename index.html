osm-to-3dm converter

Convert OpenStreetMap building footprints into Rhino .3dm volumes using a lightweight Python utility.

Prefer a graphical workflow? Run python web_converter.py and open http://127.0.0.1:8000/ to use the drag-and-drop converter in your browser.

Install the dependency

Both the command-line converter and the local web interface use the rhino3dm package. Install it with: 

pip install rhino3dm

Run the command-line converter

  1. Export an .osm file that includes building or building:part features from openstreetmap.org.
  2. Convert the export into a Rhino file: 
    python osm_to_3dm.py input.osm output.3dm
  3. Optional flags such as --default-height and --level-height let you tune how missing attributes are interpreted.

The converter keeps building footprints, projects them to a local tangent plane to minimise distortion, and extrudes them using height, min_height, or building:levels tags.

Use the local drag-and-drop server

  1. Start the server: 
    python web_converter.py --port 8000
  2. Open http://127.0.0.1:8000/ in your browser.
  3. Drop an exported .osm file to receive the converted .3dm.
